# MDTI (MaD-Type-Indicator) 人格测试

一个不太正经的人格测试小游戏，由 OopsLab 制作。

通过 16 个尴尬、离谱、但又有点真实的小场景，测出你的默认社交反应类型。它不是心理测评，更像是一面会吐槽的哈哈镜。

## 在线试玩

https://faketina.github.io/MDTI-Test/

## 你会得到什么

- 一个四字母 MDTI 人格代码
- 一个有点离谱的人格称号
- 样本占比与隐藏稀有人格
- 动物画像与植物画像
- 朋友锐评和出厂设置
- 可复制的文字结果
- 可分享的结果图片

## 隐私说明

本项目为纯前端页面，不需要登录，也不会收集个人信息。
